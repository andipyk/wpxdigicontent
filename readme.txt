== Description ==

DigiContent seamlessly integrates OpenAI GPT-4 and Anthropic Claude models into your WordPress editor for AI-powered content generation.

= Key Features =

* Dual AI Integration - Support for both OpenAI GPT-4 and Anthropic Claude
* Secure API Management - Enterprise-grade encryption for API keys
* Template System - Create and manage reusable content templates
* Variable Support - Dynamic content generation with custom variables
* Post Editor Integration - Generate content directly while editing
* Debug Tools - Built-in logging and troubleshooting capabilities
* Clean UI/UX - Modern, intuitive interface

= Pro Features =
* Batch Content Generation
* Advanced Template Management
* Priority Support
* Custom AI Model Parameters

= Privacy Considerations =

This plugin:
* Requires API keys from OpenAI and/or Anthropic
* Sends content prompts to these external services
* Does not store generated content on external servers
* Encrypts all API keys and sensitive data
* Includes options to configure data handling

For more details, visit:

* [Support](https://digikuy.com/support)

== Installation ==

1. Upload 'digicontent' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu
3. Go to Settings > DigiContent
4. Enter your API keys
5. Start generating content!

= API Keys Setup =

1. Get API keys:
   * [OpenAI API Key](https://platform.openai.com/api-keys)
   * [Anthropic API Key](https://console.anthropic.com/settings/keys)
2. Enter keys in Settings > DigiContent
3. Test connection

== Frequently Asked Questions ==

= Which AI models are supported? =

* OpenAI: GPT-4 Turbo
* Anthropic: Claude 3 Sonnet

= Are API keys securely stored? =

Yes! We use WordPress's encryption standards with additional AES-256 encryption.

= How do templates work? =

Templates use variables like ((topic)) that get replaced with your input.

= What about API costs? =

API usage is billed directly by OpenAI/Anthropic. We don't charge additional fees.
