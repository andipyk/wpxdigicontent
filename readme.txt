== Description ==

DigiContent is a powerful AI content generator that integrates OpenAI GPT-4 and Anthropic Claude models into your WordPress editor. Create engaging blog posts, product descriptions, and more with customizable templates and variables.

= Features =

* Integration with OpenAI GPT-4 and Anthropic Claude
* Customizable content templates
* Variable support for dynamic content
* Secure API key management
* Debug logging for troubleshooting
* Clean and intuitive user interface

= Requirements =

* WordPress 6.0 or higher
* PHP 8.0 or higher
* OpenAI or Anthropic API key

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/digicontent` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > DigiContent to configure your API keys
4. Start generating content from the post editor!

== Frequently Asked Questions ==

= Where do I get API keys? =

You can obtain API keys from:
* OpenAI: https://platform.openai.com/api-keys
* Anthropic: https://console.anthropic.com/settings/keys

= Is my API key secure? =

Yes! All API keys are encrypted before being stored in the database using WordPress's encryption standards.

= How do I create templates? =

Navigate to Settings > DigiContent and use the template manager to create new templates. You can use variables in the format ((variable_name)) that will be replaced with user input.
